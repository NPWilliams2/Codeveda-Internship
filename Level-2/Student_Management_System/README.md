# Student Management System

## Description
A console based Student Management System developed in C++ as part of the Codveda C/C++ Development Internship.

The application allows users to manage student records through a menu-driven interface. Student data is stored in a text file, allowing records to persists between program executions.

## Featured 
- Add new student records
- Display all student records
- Search student by ID
- Update existing student records
- Delete student records
- Save records to a file
- Load records from a file
- Input validation for IDs and grades
- Duplicate ID detection
- Menu-driven user interface

## Technologies Used
- C++
- Visual Studio
- File I/O

## Concepts Demonstrated
- Structures (struct)
- Vectors

- CRUD Operations
