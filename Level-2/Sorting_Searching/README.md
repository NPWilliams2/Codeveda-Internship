# Sorting and Searching Algorithms

## Description

A console-based Sorting and Searching Algorithms application developed in C++ as part of the Codveda C/C++ Development Internship.

The program allows users to enter a custom set of numbers, sort the data using the Bubble Sort algorithm, search for values using both Linear Search and Binary Search, and compare the time complexity of each algorithm.

## Features
- Enter a custom array of numbers
- Display entered numbers
- Bubble sort implementation
- Linear search implementation
- Binary search implementation
- Display algorithm time complexities
- Input validation
- Menu-driven interface
- Modular C++ structure using header and source files
